Primary action button for 3drizei — rounded pill, warm terracotta, gentle press-shrink. Use for any tappable action; reach for it before raw `<button>`.

```jsx
<Button variant="primary" size="lg" iconLeft={<i data-lucide="gift" />}>
  Encomendar agora
</Button>
```

Variants: `primary` (terracotta), `secondary` (copper), `outline`, `ghost`, `inverse` (ink). Sizes `sm | md | lg`. Props: `full` (100% width), `iconLeft` / `iconRight`, `disabled`.
